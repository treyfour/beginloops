## Suggested next step
Create `app/api/webhooks/route.ts` with a POST handler that reads and logs the incoming JSON payload:

```ts
// app/api/webhooks/route.ts
import { NextRequest, NextResponse } from 'next/server';

export async function POST(req: NextRequest) {
  const body = await req.json();
  console.log('Webhook received:', JSON.stringify(body, null, 2));
  return NextResponse.json({ received: true });
}
```

This is the foundation of every webhook integration — services like GitHub, Stripe, or Clerk send HTTP POST requests to a URL you provide, and this handler is what receives them. Once you can log the payload, you can start acting on specific event types (e.g. `payment.succeeded`, `push`). Run `next dev` and use a tool like [Webhook.site](https://webhook.site) or `ngrok` to forward real events to your local machine.
