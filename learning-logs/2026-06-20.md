## Suggested next step
Create `app/api/webhooks/route.ts` with a POST handler that logs incoming JSON payloads:

```ts
// app/api/webhooks/route.ts
import { NextRequest, NextResponse } from 'next/server';

export async function POST(req: NextRequest) {
  const body = await req.json();
  console.log('Webhook received:', JSON.stringify(body, null, 2));
  return NextResponse.json({ received: true });
}
```

This is how external services (GitHub, Stripe, Vercel) push events *to* your app — the opposite direction from your heartbeat, which you send *out*. Webhooks are the foundation of event-driven architectures: instead of polling an API every few seconds to check for changes, the service calls your endpoint the moment something happens. Once this route exists you can point any webhook-capable service at `https://your-app.vercel.app/api/webhooks` and start receiving real events.
